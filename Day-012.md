## ✅ Problems Solved

### 1. Valid Palindrome

**Platform:** LeetCode (LC 125)

**Approach:**

* Used two pointers from both ends.
* Ignored non-alphanumeric characters and compared the remaining characters case-insensitively.

**Time Complexity:** O(n)

**Space Complexity:** O(1)

---

### 2. Longest Common Prefix

**Platform:** LeetCode (LC 14)

**Approach:**

* Compared characters column by column across all strings.
* Stopped when a mismatch was found.

**Time Complexity:** O(n × m)

**Space Complexity:** O(1)

---

### 3. Valid Anagram

**Platform:** LeetCode (LC 242)

**Approach:**

* Counted character frequencies using a Hash Map (or an array for lowercase letters).
* Compared the frequency counts of both strings.

**Time Complexity:** O(n)

**Space Complexity:** O(n)

---

### 4. Reverse Words in a String

**Platform:** LeetCode (LC 151)

**Approach:**

* Split the string into words.
* Removed extra spaces.
* Reversed the order of the words and joined them back.

**Time Complexity:** O(n)

**Space Complexity:** O(n)

---

## 🧠 Key Learnings

* Two Pointers simplify many string problems.
* Hash Maps are useful for character frequency and mapping.
* Always consider edge cases like empty strings, spaces, and case sensitivity.
* String manipulation is a core topic in coding interviews.
