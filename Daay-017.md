## ✅ Problems Solved

### 1. Leaders in an Array

**Platform:** GeeksforGeeks

**Approach:**

* Traversed the array from right to left.
* Maintained the maximum element seen so far and recorded leaders.

**Time Complexity:** O(n)

**Space Complexity:** O(1)

---

### 2. Rotate Image (90° Clockwise)

**Platform:** LeetCode (LC 48)

**Approach:**

* Transposed the matrix.
* Reversed each row to achieve a 90° clockwise rotation in-place.

**Time Complexity:** O(n²)

**Space Complexity:** O(1)

---

### 3. Set Matrix Zeroes

**Platform:** LeetCode (LC 73)

**Approach:**

* Used the first row and first column as markers to indicate which rows and columns should be set to zero.
* Performed the update in-place with constant extra space.

**Time Complexity:** O(m × n)

**Space Complexity:** O(1)

---

## 🧠 Key Learnings

* Traversing from the correct direction can simplify array problems.
* Many matrix problems can be solved in-place using matrix properties.
* Using existing matrix rows and columns as markers is a common optimization technique.
* Arrays and matrices are among the most frequently tested interview topics.
