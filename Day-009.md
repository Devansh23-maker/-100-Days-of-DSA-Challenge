📚 Topics Covered

---

✅ Problems Solved

1. Valid Parentheses

Platform: LeetCode

Approach:

- Used a stack to push opening brackets.
- Popped and matched brackets whenever a closing bracket was encountered.

Time Complexity: O(n)

Space Complexity: O(n)

---

2. Next Greater Element I

Platform: LeetCode

Approach:

- Used a Monotonic Decreasing Stack to efficiently find the next greater element for each number.

Time Complexity: O(n)

Space Complexity: O(n)

---

3. Min Stack

Platform: LeetCode

Approach:

- Maintained an additional stack to keep track of the minimum element at every step.

Time Complexity:

- Push: O(1)
- Pop: O(1)
- Top: O(1)
- GetMin: O(1)

Space Complexity: O(n)

---

4. Daily Temperatures

Platform: LeetCode

Approach:

- Used a Monotonic Decreasing Stack to find the next warmer day for each temperature.

Time Complexity: O(n)

Space Complexity: O(n)

---

🧠 Key Learnings

- Stacks follow the Last In, First Out (LIFO) principle.
- Monotonic Stacks efficiently solve Next Greater/Smaller Element problems.
- Stack-based solutions often replace nested loops with linear-time algorithms.
- Recognizing when to use a stack is an important interview skill.
