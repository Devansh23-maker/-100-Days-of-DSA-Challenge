## ✅ Problems Solved

### 1. 3Sum

**Platform:** LeetCode (LC 15)

**Approach:**

* Sorted the array.
* Fixed one element and used two pointers to find pairs whose sum equaled the target while skipping duplicates.

**Time Complexity:** O(n²)

**Space Complexity:** O(1)

---

### 2. 4Sum

**Platform:** LeetCode (LC 18)

**Approach:**

* Sorted the array.
* Fixed two elements and applied the Two Pointer technique to find valid quadruplets while avoiding duplicates.

**Time Complexity:** O(n³)

**Space Complexity:** O(1)

---

### 3. Pascal's Triangle

**Platform:** LeetCode (LC 118)

**Approach:**

* Built each row using the values from the previous row.
* The first and last elements of every row were initialized to `1`.

**Time Complexity:** O(n²)

**Space Complexity:** O(n²)

---

## 🧠 Key Learnings

* Sorting combined with Two Pointers is a powerful pattern for solving sum-related problems.
* Skipping duplicate elements is essential to avoid repeated answers.
* Pascal's Triangle demonstrates dynamic construction using previously computed values.
* These problems reinforce array manipulation, optimization, and pattern recognition.
