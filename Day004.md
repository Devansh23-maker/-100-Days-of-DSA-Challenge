📚 Topics Covered

- Fixed-Size Window
- Variable-Size Window
- Subarrays & Substrings
- Window Expansion and Shrinking

---

✅ Problems Solved

1. Maximum Average Subarray I

Platform: LeetCode

Approach:

- Used a fixed-size sliding window.
- Calculated the first window sum, then updated it by removing the left element and adding the new right element.

Time Complexity: O(n)

Space Complexity: O(1)

---

2. Maximum Number of Vowels in a Substring of Given Length

Platform: LeetCode

Approach:

- Maintained a fixed-size window.
- Counted vowels entering and leaving the window to update the answer efficiently.

Time Complexity: O(n)

Space Complexity: O(1)

---

3. Longest Substring Without Repeating Characters

Platform: LeetCode

Approach:

- Used a variable-size sliding window with a Hash Set.
- Expanded the window while characters were unique and shrank it when duplicates appeared.

Time Complexity: O(n)

Space Complexity: O(n)

---

🧠 Key Learnings

- Sliding Window converts many O(n²) solutions into O(n).
- Fixed-size windows are useful when the window length is constant.
- Variable-size windows help solve optimization problems involving conditions.
- Hash Sets and Hash Maps are commonly used with Sliding Window.
