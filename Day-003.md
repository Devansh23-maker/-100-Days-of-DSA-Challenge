✅ Problems Solved

1. Valid Palindrome

Platform: LeetCode

Approach:

- Used two pointers starting from both ends of the string.
- Ignored non-alphanumeric characters and compared characters until the pointers met.

Time Complexity: O(n)

Space Complexity: O(1)

---

2. Merge Sorted Array

Platform: LeetCode

Approach:

- Started from the end of both arrays.
- Compared elements and placed the larger one at the end of the merged array.

Time Complexity: O(m + n)

Space Complexity: O(1)

---

3. Move Zeroes

Platform: LeetCode

Approach:

- Used two pointers to move all non-zero elements to the front while maintaining their relative order.
- Filled the remaining positions with zeros.

Time Complexity: O(n)

Space Complexity: O(1)

---

🧠 Key Learnings

- Two Pointers can optimize many brute-force solutions.
- In-place algorithms reduce extra memory usage.
- Pointer movement depends on the problem's conditions.
- This pattern is widely used in coding interviews.
