✅ Problems Solved

1. Find All Anagrams in a String

Platform: LeetCode (LC 438)

Approach:

- Used a fixed-size Sliding Window with character frequency counts.
- Compared frequencies to identify all anagram starting indices.

Time Complexity: O(n)

Space Complexity: O(26)

---

2. Longest Palindromic Substring

Platform: LeetCode (LC 5)

Approach:

- Expanded around each character (and between characters) to find the longest palindrome.

Time Complexity: O(n²)

Space Complexity: O(1)

---

🧠 Key Learnings

- Sliding Window is powerful for substring-related problems.
- Expand Around Center is an effective technique for palindrome problems.
- Choosing the correct pattern often leads to an optimal solution.
- Always analyze both time and space complexity.
