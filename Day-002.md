✅ Problems Solved

1. Two Sum

Platform: LeetCode

Approach:

- Used a Hash Map to store previously seen numbers.
- Checked whether the required complement existed before inserting the current number.

Time Complexity: O(n)

Space Complexity: O(n)

---

2. Best Time to Buy and Sell Stock

Platform: LeetCode

Approach:

- Maintained the minimum stock price seen so far.
- Calculated the maximum possible profit at each step.

Time Complexity: O(n)

Space Complexity: O(1)

---

3. Contains Duplicate

Platform: LeetCode

Approach:

- Used a Hash Set to track visited elements.
- Returned "True" immediately when a duplicate was found.

Time Complexity: O(n)

Space Complexity: O(n)

---

🧠 Key Learnings

- Hash Maps can reduce brute-force solutions from O(n²) to O(n).
- Keeping track of the minimum value seen so far is an effective optimization pattern.
- Hash Sets are useful for detecting duplicates efficiently.
- Always analyze both time and space complexity before finalizing a solution.
