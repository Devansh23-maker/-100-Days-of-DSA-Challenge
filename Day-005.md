## 📅 Date

July 9, 2026

## 🎯 Goal

Learn how Hash Maps and Hash Sets can optimize array and string problems by enabling constant-time lookups.

---

## 📚 Topics Covered

* Hash Map (Dictionary)
* Hash Set
* Frequency Counting
* Key-Value Pair Storage
* Fast Lookup

---

## ✅ Problems Solved

### 1. Valid Anagram

**Platform:** LeetCode

**Approach:**

* Counted the frequency of each character using a Hash Map.
* Compared the frequency counts of both strings.

**Time Complexity:** O(n)

**Space Complexity:** O(n)

---

### 2. Ransom Note

**Platform:** LeetCode

**Approach:**

* Stored character frequencies from the magazine.
* Checked if each character in the ransom note was available in sufficient quantity.

**Time Complexity:** O(m + n)

**Space Complexity:** O(1) *(or O(26) for lowercase English letters)*

---

### 3. Intersection of Two Arrays

**Platform:** LeetCode

**Approach:**

* Stored one array in a Hash Set.
* Traversed the second array and collected common unique elements.

**Time Complexity:** O(m + n)

**Space Complexity:** O(m)

---

## 🧠 Key Learnings

* Hash Maps provide average O(1) lookup, insertion, and deletion.
* Frequency counting is one of the most common Hash Map applications.
* Hash Sets efficiently detect duplicates and common elements.
* Choosing the right data structure can significantly improve performance.
