✅ Problems Solved

1. Reverse an Array

Platform: GeeksforGeeks

Approach:

- Swap elements from both ends.
- Continue until the pointers meet.

Time Complexity: O(n)

Space Complexity: O(1)

---

2. Count Frequency of an Element

Platform: GeeksforGeeks

Approach:

- Traverse the array once.
- Count occurrences of the target element.

Time Complexity: O(n)

Space Complexity: O(1)

---

3. Majority Element

Platform: GeeksforGeeks

Approach:

- Use the Boyer–Moore Voting Algorithm to find the majority element efficiently.

Time Complexity: O(n)

Space Complexity: O(1)

---

🧠 Key Learnings

- Arrays provide constant-time access using indices.
- Always look for opportunities to reduce nested loops.
- Analyze every solution based on time and space complexity.
- Two-pointer techniques are useful for many array problems.
