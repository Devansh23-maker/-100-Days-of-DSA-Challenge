## ✅ Problems Solved

### 1. Reverse String

**Platform:** LeetCode (LC 344)

**Approach:**

* Used two pointers from both ends of the array.
* Swapped characters until the pointers met.

**Time Complexity:** O(n)

**Space Complexity:** O(1)

---

### 2. Reverse String II

**Platform:** LeetCode (LC 541)

**Approach:**

* Processed the string in blocks of `2k`.
* Reversed the first `k` characters of every block.

**Time Complexity:** O(n)

**Space Complexity:** O(1)

---

## 🧠 Key Learnings

* Two Pointers are highly effective for string reversal problems.
* Careful index manipulation is essential when working with strings.
* Always consider edge cases such as empty strings and single-character inputs.
* Practicing simple string problems builds a strong foundation for advanced string algorithms.
